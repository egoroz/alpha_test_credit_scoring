# alpha_test_credit_scoring

Тестовое задание от Альфа-Банка для поступления в магистратуру ФПМИ МФТИ.
Решение находится только в `credit_scoring_alpha.ipynb`.

## Задача

Оценить вероятность дефолта клиента по его кредитной истории. Метрика — ROC-AUC.

## Подход

Ансамбль из нескольких разнородных моделей: часть работает с агрегатами по всей истории клиента, часть - напрямую с последовательностью продуктов. Предсказания моделей объединяются, после чего уточняются за счёт самостоятельной разметки тестовых данных. Обученные модели и их предсказания кэшируются, поэтому повторный запуск переиспользует посчитанное.
