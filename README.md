# alpha_test_credit_scoring

Тестовое задание Альфа-Банка для поступления в магистратуру ФПМИ МФТИ.

## Задача

Оценить вероятность дефолта клиента по его кредитной истории. Метрика ROC-AUC.

## Подход

Ансамбль разнородных моделей: часть работает с агрегатами по всей истории клиента, а другая часть напрямую с последовательностью кредитных продуктов. Предсказания объединяются стекингом и уточняются за счёт самостоятельной разметки тестовых данных. Расчёт на GPU (A100).

## Структура

Решение разбито на три ноутбука:

- `credit_scoring_eda.ipynb` — EDA.
- `credit_scoring_tuning.ipynb` — подбор гиперпараметров моделей и порогов самостоятельной разметки.
- `credit_scoring_alpha.ipynb` — основное решение: обучение моделей, ансамбль, формирование сабмита.
